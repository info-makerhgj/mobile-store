# حسابات النظام

## حساب المدير (Admin)

### إنشاء حساب مدير
```bash
cd backend
npm run create:admin
```

### بيانات تسجيل الدخول
- **البريد الإلكتروني:** admin@abaad.sa
- **كلمة المرور:** admin123
- **رابط تسجيل الدخول:** http://localhost:3000/admin/login
- **لوحة التحكم:** http://localhost:3000/admin

### صلاحيات المدير
- إدارة المنتجات (إضافة، تعديل، حذف)
- إدارة الطلبات
- عرض العملاء
- الإحصائيات والتقارير
- الإعدادات

---

## حساب العميل (Customer)

### إنشاء حساب عميل تجريبي
```bash
cd backend
npm run create:customer
```

### بيانات تسجيل الدخول
- **البريد الإلكتروني:** customer@example.com
- **كلمة المرور:** customer123
- **رابط تسجيل الدخول:** http://localhost:3000/login
- **صفحة الحساب:** http://localhost:3000/account

### صلاحيات العميل
- تصفح المنتجات
- إضافة منتجات للسلة
- إتمام الطلبات
- عرض طلباتي
- إدارة العناوين
- تعديل الملف الشخصي

---

## الفرق بين الحسابات

| الميزة | المدير | العميل |
|--------|--------|---------|
| تسجيل الدخول | `/admin/login` | `/login` |
| الصفحة الرئيسية | `/admin` | `/` |
| إدارة المنتجات | ✅ | ❌ |
| إدارة الطلبات | ✅ | عرض طلباته فقط |
| الشراء | ❌ | ✅ |
| الإحصائيات | ✅ | ❌ |

---

## ملاحظات مهمة

1. **حماية صفحات المدير:** جميع صفحات `/admin/*` محمية ولا يمكن الوصول إليها إلا بحساب مدير
2. **فصل تام:** حساب المدير منفصل تماماً عن حساب العميل
3. **تسجيل خروج آمن:** عند تسجيل الخروج يتم حذف التوكن والتوجيه للصفحة المناسبة
4. **التحقق التلقائي:** يتم التحقق من صلاحيات المستخدم عند كل طلب

---

## إنشاء حسابات جديدة

### إنشاء مدير جديد
يمكنك تعديل ملف `backend/create-admin.ts` وتغيير البيانات ثم تشغيل:
```bash
npm run create:admin
```

### إنشاء عميل جديد
- من خلال صفحة التسجيل: http://localhost:3000/register
- أو تعديل `backend/create-customer.ts` وتشغيل:
```bash
npm run create:customer
```
